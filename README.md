# Assembler and Linker

This is an assembler and linker written in C for a fictional assembly language called LC2K. LC2K was written by staff at the University of Michigan and is made up of simple commands that allow for loading a word, storing a word, addition, branching, halting, etc.

The full description of LC2K can usually be found [here](https://eecs370.github.io/project_1_spec/). As a quick reference, a short program multiplying two positive 15-bit numbers in LC2K is included in `/multiplication`.
